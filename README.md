# ✨ Gemini Image Bot

A powerful Discord bot that generates AI images using **Gemini 2.0 Flash** — Google's blazing-fast, multi-modal model. Type your prompt, and the bot returns stunning visuals in seconds!

---

## 🚀 Features

- 🎨 **AI Image Generation** powered by Gemini 2.0 Flash
- ⚡ **Fast response time** with minimal latency
- 💬 **Simple and intuitive Discord commands**
- 🖼️ **Direct image previews** in your Discord server

---

! a futuristic city 
!a cool person
